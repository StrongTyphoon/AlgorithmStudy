## 그래프의 표현
### 그래프 
- 사물을 정점(Vertex, Node)과 간선(Edge)으로 나타내기 위한 도구
- 두가지 방법
1. 인접행렬 : 2차원 배열 사용
2. 인접리스트 : 연결리스트 이용

### 인접행렬
- 그래프를 **2차원 배열**로 표현한다.
- 무방향 무가중치 그래프 
- 방향 가중치 그래프

### 인접리스트
- 그래프를 리스트 형태로 표현
- 무방향 무가중치 그래프
```
graph = [
  [1,2],
  [0,2],
  [0,1,3],
  [2]
]
```
- 방향 가중치 그래프
```
graph = [
  [(2,7)],
  [(0,3), (2,8)],
  [(1,8)],
  [(2,4)]
]
```

### 그래프의 시간복잡도
- 인접행렬 O(V^2) 공간효율도는 떨어지나, 연결 여부를 O(1)로 **낮은 시간복잡도** 알 수 있다
- 인접리스트는 **공간효율도O(V+E)는 높**으나, 연결 여부 확인에 O(V) 만큼의 시간 복잡도를 가진다

### 보통 인접리스트를 사용한다.
- 노드의 연결이 적기 때문
